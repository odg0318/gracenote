Sinatra Sample
==============

REST API auth module

Test environment
----------------

* Ubuntu 16.04 server
* Ruby 2.3.0p0

Get started!
------------
	$ sudo apt-get install ruby gem
	$ sudo gem install bundler
	$ git clone https://github.com/odg0318/gracenote
	$ cd gracenote
	$ bundle install
	$ ruby initialize.rb
	
Configuration
-------------

You can see config.yaml.

API List
--------
	GET /
	POST /login
	POST /logout
	GET /members
	PUT /members
